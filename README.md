<h3>실행 방법</h3>

- Repository Clone

```
git clone -b https://github.com/dnd-side-project/dnd-7th-6-frontend.git hot6
cd hot6
```

- Download node_modules

```
npm install || yarn
```

- Start Emulator

- Start App on Android

```
yarn android
```

- Start App on iOS

```
cd ios
pod install #M1 Mac | arch -x86_64 pod install
yarn ios
```
